# Twixt 멀티플레이 프로젝트

간단한 **2인 온라인 Twixt** 예제입니다. VS Code에서 바로 열어서 실행할 수 있게 만들었습니다.

## 포함 기능
- 2인 실시간 멀티플레이 (Socket.IO)
- 방 코드로 같은 방 접속
- 관전자 접속 가능
- Twixt 기본 규칙 반영
  - 말 1개씩 번갈아 두기
  - 나이트 이동 거리의 같은 색 말끼리 자동 링크
  - 링크 교차 금지
  - Red: 위↔아래 연결 / Blue: 좌↔우 연결
  - 상대 진영 테두리 금지
- **파이 룰(swap rule)** 지원
- 다시 시작 버튼 지원

## 실행 방법
```bash
npm install
npm run dev
```

그 다음 브라우저에서 아래 주소를 여세요.
```bash
http://localhost:3000
```

## 같은 컴퓨터에서 테스트
브라우저 창 2개를 열고 같은 방 코드로 접속하면 됩니다.

## 다른 사람과 테스트
같은 네트워크라면 서버를 실행한 컴퓨터의 IP로 접속하면 됩니다.
예:
```bash
http://192.168.0.10:3000
```

외부 인터넷으로 하려면 Render, Railway, Fly.io, VPS 같은 곳에 올리면 됩니다.

## 파일 구조
- `server.js` : 서버 + 실시간 통신 + 게임 상태 관리
- `public/index.html` : 화면
- `public/styles.css` : 스타일
- `public/app.js` : 클라이언트 게임 로직

## 참고
이 구현은 **실용적인 입문용 버전**입니다.
공식/전통 규칙에서는 자신의 링크 제거 후 재배치도 허용되는 설명이 있는데, 이 버전은 **자동 링크 생성 방식**으로 단순화했습니다. TwixT 규칙의 핵심인 나이트 링크, 교차 금지, 목표 연결은 반영했습니다.
